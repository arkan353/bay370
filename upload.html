📤 Загрузка через curl

# Простая загрузка файла
curl -F "file=@photo.jpg" {{site_url}}/api/upload

# Смена имени в облаке (флаг необязательный)
curl -F "file=@photo.jpg" -F "name=myphoto.jpg" {{site_url}}/api/upload

# Пример ответа:
{"ok": true, "name": "photo.jpg", "download_url": "http://.../dl/abc/0"}

📥 Скачивание через wget

# После загрузки вы получите ссылку вида:
wget {{site_url}}/dl/abc123/0

# Файл сохранится с оригинальным именем
# Ссылка действительна 1 час

💡 Как это работает

  1. При первом посещении сайт выдаёт вам персональный ключ (куки)
  2. Загруженные файлы привязываются к этому ключу — чужие файлы вам не видны
  3. После загрузки вы получаете ссылку на файл, которой можно делиться
  4. Ссылка ведёт на защищённое облачное хранилище и работает 1 час
  5. Скачивать можно из браузера или через wget
